## Perguntas de Negócio

Após a realização da análise exploratória de dados (EDA), esta etapa tem como objetivo responder às principais perguntas de negócio, utilizando consultas SQL para gerar insights relevantes para a empresa.

1. Qual funcionário realizou o maior volume de vendas?
2. Quais clientes apresentam maior frequência de compra?
3. Quais produtos geram maior receita para a empresa?
4. Como a receita se comporta ao longo dos períodos analisados?
